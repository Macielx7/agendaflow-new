# Dr. João Silva — Landing Page Premium

Landing page de alta conversão para clínica odontológica premium (lentes de contato dental e implantes).

## Tecnologias

- Next.js 14 (App Router)
- React 18
- Framer Motion
- SwiperJS
- CSS Modules
- Lucide React & React Icons

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Personalização

Edite `utils/constants.js` para alterar:

- WhatsApp e Instagram
- Endereço e horários
- Textos e depoimentos
- Imagens (URLs ou arquivos em `public/`)

## Build

```bash
npm run build
npm start
```
